## Team Stories

- As a Super user I want to register my business account.
- As a Super user I want to have access over everything and manage everything.
- As a Super user I want to approve and disapprove my admins and employees.
- As a Super user I want to update profiles.
- As an Admin user I want to register to an approved business and get approved by super user.
- As an Admin user I want to approve and disapprove employees.
- As an Admin user I want to have access over everything except managing profiles of other admins and changing business crucial details like GSTIN or name though I can update location.
- As an Employee user I register to an approved business and get approved by admin as well as Super user.
- I can manage products, orders, rma, discount and courses but I cannot do what Admin and Super users can do.
